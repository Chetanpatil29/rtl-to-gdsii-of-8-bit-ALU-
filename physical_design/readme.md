# Physical Design

This folder contains the backend ASIC implementation stages performed during the RTL-to-GDSII flow using Cadence Innovus and Virtuoso.

## Physical Design Flow

- Floorplanning
- Pad Placement
- Power Planning
- Standard Cell Placement
- Clock Tree Synthesis (CTS)
- Routing
- Static Timing Analysis (STA)
- DRC/LVS Verification
- GDSII Generation

Each stage is documented separately with screenshots and a brief explanation.
