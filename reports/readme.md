| timing_report.md | Static Timing Analysis (STA) |
| area_report.md | Cell area utilization |
| power_report.md | Estimated power consumption |
| lec_report.md | Logical Equivalence Check |
| gls_report.md | Gate-Level Simulation |
| drc_lvs_report.md | Physical verification summary |
